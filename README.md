# Pokedex

###Description:
This application allows users to do a quick search on their _favorite_ Pokemons, extracted from an external API. **150 Pokemons** are featured, together with their height and weight details, as well as high definition pictures, arguably ensuring at least ten favorites for every user.

###User methodology:
The search can be run through the search box on the header of the page, or by clicking or touching on the desired name seen on the list of Pokemons in the body of the page. The above mention details would then appear automatically. The user can return to the beginning of the app anytime, just by clicking on the **POKEDEX** logo in the left corner of the pages' header.

###Technologies used:

- HTML
- CSS
- JavaScript
- jQuery
- Bootstrap
